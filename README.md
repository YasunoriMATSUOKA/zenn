## Cheatsheet

### Create New Article

```sh
npx zenn new:article --slug 記事のスラッグ --title タイトル --type idea --emoji ✨
```

### Create New Book

```sh
$ npx zenn new:book
# 本のslugを指定する場合は以下のようにします。
# npx zenn new:book --slug ここにスラッグ
```
